1. PROJECT OVERVIEW

The SME Food Project Admin Page is a dynamic web-based administrative interface developed to empower restaurant owners within the SME Food ecosystem. This platform enables owners to manage their menus, receive orders from the SME Food mobile app, and gain insights through analytics, thereby optimizing operational efficiency and customer satisfaction. My role encompassed the full-stack development of this admin page, focusing on creating a seamless experience for restaurant owners to control their business processes effectively.

2.MY ROLE

As a Full-Stack Developer, I spearheaded the development of the SME Food Project Admin Page. My responsibilities spanned across designing the user interface, implementing backend logic, integrating real-time order processing, and incorporating analytics functionalities. Key contributions include:

  • Crafting an intuitive and responsive UI that allows restaurant owners to easily update menus, including special offers and new items.
  • Developing a robust backend system to handle real-time order notifications, enabling owners to efficiently manage incoming orders, process them, and update their statuses (open, in-process, completed).
  • Integrating analytics tools to provide restaurant owners with actionable insights into customer preferences, order trends, and operational efficiencies.
  • Ensuring secure and reliable communication between the SME Food mobile app and the admin page through encrypted APIs.

3. TECHNICAL HIGHLIGHTS

These are the technical highlights  

  • Utilized React along with Ant Design for the front-end development, creating a modern and user-friendly interface.
  • Implemented Node.js and Express for the backend, facilitating scalable API development and data management.
  • Integrated MongoDB for database management, storing menu items, order details, and analytics data efficiently.
  • Incorporated Grafana for visualizing analytics data, offering restaurant owners valuable insights into business performance.
  • Applied JWT (JSON Web Tokens) for secure authentication and authorization, ensuring data integrity and confidentiality.

4. OUTCOMES

These are the project outcomes  

  • Streamlined menu management and order processing for restaurant owners, significantly reducing operational complexities.
  • Enhanced customer experience through real-time order updates and efficient service delivery.
  • Provided restaurant owners with critical business insights, aiding in strategic decision-making and business growth.
